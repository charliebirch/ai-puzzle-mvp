# 2026-03-28 — Wizard Pipeline Rebuild

## What We Built

Rebuilt the entire web app from a single-form "upload and wait" flow into a **5-step interactive wizard** with a proven identity-preserving pipeline.

### The Wizard (5 Steps)
1. **Upload** — Photo + person description (age, gender, ethnicity, hair, skin tone). File type validation (JPEG/PNG/HEIC only). HEIC auto-converted. Quality checks: face detection, blur detection, size warnings.
2. **Background Removal** — lucataco/remove-bg ($0.01). Shows before/after.
3. **Character Generation** — Kontext Max ($0.08). Pixar-style transform with identity preservation.
4. **Costume** — Kontext Max ($0.08). Themed outfit (village adventurer).
5. **Scene & Composite** — FLUX 2 Pro scene ($0.08) + 3 Kontext Max candidates ($0.24) + Real-ESRGAN upscale ($0.002). Auto-picks best via quality scorer. Shows single final result.

**Total cost: ~$0.49 per puzzle.**

### Key Discovery: Identity Preservation

The biggest learning of the session. The AI model (Kontext Max) **does not reliably preserve ethnicity, skin tone, or hair texture** from the photo alone. Tested with Joel (Black, afro hair) — model changed him to look Asian with slicked-back hair.

**Fix:** The character prompt now explicitly states: *"This person is {subject} — preserve their exact skin tone, hair texture, hair color, hairstyle, and face shape."* The compositing prompt adds: *"Do not change the character's face, skin tone, hair, or ethnicity."*

This dramatically improved results for Joel. The subject description is built from structured dropdowns (ethnicity, skin tone, hair color, hair style) plus auto-detected skin tone from the photo.

### Auto-Detect Skin Tone

OpenCV face detection → sample forehead region → LAB color space → map L channel to skin tone label. Runs on the prepared PNG (not HEIC original — that was a bug we fixed). Falls back gracefully if no face detected.

### Multi-Seed + Quality Scoring

Step 5 generates 3 composite candidates with different seeds, scores each with the 11-metric puzzle quality scorer, and picks the highest. Scores typically 86-98/100. The winner gets upscaled 2x via Real-ESRGAN (anime model preferred, general as fallback — anime model is currently broken on Replicate).
